# MemoriesCommand

## Overview
Command collection that registers the `memories` command and its subcommands for memory management.

## Constructor Descriptions
- `MemoriesCommand()`: Registers subcommands for clearing, capacity changes, level queries, and unlocking.

## Method Descriptions
- `MemoriesCommand()`: Configures the command tree with `clear`, `capacity`, `level`, and `unlockAll`.

## Examples
```java
// /memories level
```
