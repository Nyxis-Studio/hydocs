Esta é a classe principal de entrada do servidor Hytale.

### Como Iniciar

Normalmente, você não precisa interagir diretamente com esta classe se estiver escrevendo um mod/plugin. O servidor chama o `main()` automaticamente.

#### Exemplo de Argumentos de Inicialização:

```bash
java -jar HytaleServer.jar --nogui --port 25565
```

**Nota Importante:** Certifique-se de configurar o `server.properties` antes de iniciar pela primeira vez.
