# AssetConstants

**Overview**
Provides expected-count constants used by asset validation and organization.
Centralizes fixed values so loaders and validators stay consistent.

**Constructors**
- Not applicable. Utility class with constants only.

**Methods**
- No methods.

**Notes**
- `EXPECTED_CHILDREN_PER_ASSET`: expected child count per asset.
- `EXPECTED_ASSETS_PER_PATH`: expected asset count per path.
- `EXPECTED_VALUES_PER_TAG`: expected value count per tag.
- `EXPECTED_ASSETS_PER_TAG`: expected asset count per tag.
